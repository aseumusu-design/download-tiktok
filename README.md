# TikTok Downloader

Paket ini sudah dipisah menjadi:

- `index.html`
- `styles.css`
- `script.js`
- `.nojekyll`

## Upload ke GitHub Pages

1. Buat repository baru di GitHub.
2. Upload semua isi folder ini ke root repository.
3. Masuk ke `Settings` lalu `Pages`.
4. Pilih branch utama, biasanya `main`.
5. Simpan dan tunggu GitHub Pages aktif.

## Catatan

Project ini adalah static site. Jika endpoint API pihak ketiga memblokir `CORS` saat dibuka dari domain GitHub Pages, maka download bisa gagal. Kalau itu terjadi, perlu ditambah proxy atau backend.
